# Version 1 - Original Portfolio Design

This is the original portfolio design created before the editorial redesign.

## Access

When running locally: `http://localhost:4321/v1`
When deployed: `https://sboulos.com/v1`

## Design Characteristics

**Original Design (v1):**
- Multi-section single-page layout
- All content on homepage (Hero, Projects, Skills, Contact)
- Grid-based project cards
- Standard spacing (4rem/6rem)
- Hash-based navigation (#about, #projects, #skills, #contact)
- Full "Shane Boulos" logo
- Traditional portfolio aesthetic

**Layout:**
- Max-width: 1200px
- Multi-column grids for projects and skills
- Alternating section backgrounds (#fafafa)
- Card-based project display with hover effects

## Purpose

This version exists as a reference for:
- Comparing design approaches
- Reviewing the original structure
- Understanding the evolution of the portfolio
- Testing different aesthetics

## Key Differences from Current Version

| Aspect | v1 (Original) | Current (Editorial) |
|--------|--------------|---------------------|
| Layout | Multi-section single page | Multi-page with routing |
| Spacing | 4rem/6rem | 9rem/12rem |
| Max-width | 1200px | 680px single column |
| Navigation | Hash links | Page routes |
| Projects | Grid of cards | Editorial case studies |
| Content | All on one page | Distributed across pages |
| Aesthetic | Traditional portfolio | Editorial/minimal |
