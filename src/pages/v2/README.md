# Version 2 - Clean Editorial Single-Page

This is the clean, minimal single-page layout without animations.

## Access

When running locally: `http://localhost:4321/v2`
When deployed: `https://sboulos.com/v2`

## Design Characteristics

**v2 Design:**
- Single-page scrolling layout
- Four sections: Hero, Work (AWJ), About, Contact
- Editorial aesthetic with generous whitespace (9-12rem)
- Single-column layout (680px max width)
- Mixed typography: Space Mono (headings/nav) + Anonymous Pro (body)
- Hash-based navigation (#work, #about, #contact)
- Clean, minimal - no animations or background effects

## Purpose

This version preserves the clean, distraction-free layout before adding scroll animations.
